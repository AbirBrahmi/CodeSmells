# Fetch_remote_github-commits

## Description
Le script permet de demander les informations de l'utilisateur et du dépôt, puis il construit une URL pour récupérer les commits du dépôt GitHub spécifié.

## Paramètres
- `user` : Nom de l'organisation ou de l'utilisateur.
- `repository` : Nom du dépôt.

## Explication
L'utilisateur et le dépôt peuvent être trouvés dans l'URL du dépôt GitHub. Le nom de l'organisation ou de l'utilisateur se trouve avant le nom du dépôt.

### Exemple
Pour le dépôt [https://github.com/OrchardCMS/OrchardCore](https://github.com/OrchardCMS/OrchardCore) :
- `OrchardCMS` est l'organisation ou l'utilisateur.
- `OrchardCore` est le nom du dépôt.
